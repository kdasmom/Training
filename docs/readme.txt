INSTRUCTIONS TO GENERATE DOCS FOR NEXUSPAYABLES

PHP
- Setup PHPDocumentor 2 (http://www.phpdoc.org/)
- Easiest way to setup is via PEAR (see web site)
- Make sure PHPdocumentor 2 binary is in your PATH, open a command prompt and navigate to
  the directory where this readme.txt file is located, run the command phpdoc, and the docs will
  be generated inside a php subfolder

Javascript
- Setup JSDuck (https://github.com/senchalabs/jsduck)
- If you're using Windows, you can just download a binary
- Make sure JSDuck binary is in your PATH, then run the generateJSDocs.bat file located in the
  same directory as this readme.txt file and the docs will be generated inside a js subfolder
